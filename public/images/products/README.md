# Images des produits

Placez ici les images de vos produits affiliés.

## Recommandations :

- **Format** : JPG ou WebP pour de meilleures performances
- **Dimensions** : 800x600px minimum pour les images principales
- **Poids** : Optimisez vos images (max 200KB idéalement)
- **Nommage** : Utilisez des noms descriptifs (ex: `aspirateur-xyz-main.jpg`)

## Structure suggérée :

```
/public/images/products/
  ├── produit-xyz-main.jpg
  ├── produit-xyz-1.jpg
  ├── produit-xyz-2.jpg
  ├── produit-abc-main.jpg
  └── ...
```

## Outils d'optimisation :

- TinyPNG (https://tinypng.com/)
- Squoosh (https://squoosh.app/)
- ImageOptim (Mac)
